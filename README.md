# FUTURE_PE_01

## Project Title
AI Website Copy Generator for Local Businesses

## Business Selected
Excel Academy Coaching Institute

## Tools Used
- ChatGPT
- Lovable
- GitHub

## Objective
The objective of this project is to generate professional, website-ready content for a local coaching institute using Prompt Engineering techniques. The project demonstrates how AI can be used to create effective marketing and website copy through iterative prompt improvement.

---

## Business Overview

*Business Name:* Excel Academy Coaching Institute

*Location:* Vijayawada, Andhra Pradesh

*Services Offered:*
- Intermediate Coaching
- JEE Preparation
- NEET Preparation
- EAMCET Coaching
- Foundation Courses

*Target Audience:*
Students of Classes 9–12 and Competitive Exam Aspirants.

---

## Prompt Versions

### Prompt Version 1

text
Create website content for a coaching institute.


### Prompt Version 2

text
Create professional website content for a coaching institute in Vijayawada offering JEE, NEET, EAMCET and Intermediate coaching.

Target students and parents.

Include Homepage, About Us, Courses and Contact sections.


### Prompt Version 3

text
Act as an expert educational website copywriter.

Create persuasive website content for Excel Academy in Vijayawada.

Focus on:
- Student success
- Experienced faculty
- Exam preparation
- Trust building
- Strong call-to-action

Generate:
- Homepage
- About Us
- Course Descriptions
- Why Choose Us
- Contact CTA

## Prompt Engineering Analysis

### Clarity

The prompts were gradually improved by adding:
- Business name
- Location
- Services offered
- Target audience
- Required website sections

This helped generate more accurate and relevant content.

### Trust

Trust-building elements were added through:
- Experienced faculty
- Student success focus
- Testimonials
- Professional tone
- Educational expertise

These elements make the website more credible and appealing to parents and students.

### Conversions

Conversion-focused improvements included:
- Strong headline
- Clear call-to-action
- Course benefits
- Student-focused messaging
- Encouraging enrollment language

These additions help motivate visitors to take action.
